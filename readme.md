## General info
This is a virtual airline created for use with the [Vatsim](https://vatsim.net) Network.
	
## Technologies
Project is created with:
* Laravel Version: 5.8
* Ipinfo Version: 1.0
* Aviation API
* Vue Js
* React Js

##Setup

```
$ composer install
$ npm install
$ php artisan preset react
$ composer require ipinfo/laravel

```
